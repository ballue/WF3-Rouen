Les formulaires :

  • La balise FORM permet de creer le formulaire.
  • name : Permet de donner un nom au formulaire
  • action: Cet attribut est obligatoire et permet de renseigner le script/ la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du formulaire. Cette page va recevoir l'ensemnle des données saisie pas mon utilisateur.
  • method : Permet de préciser quelle méthode [ GET; POST] doit être utilisé par le navigateur pour transmettre les données saisies par l'utilisateur vers la pageACTION














créer un lien en HTML

  • la balise "a" permet de créer un lien vers unatre ressource.(image, vidéos, PDF,ZIP, ...)

  • l'attribut href : Permet de définir le chemin vers le fichier à lier.

  • L'attribut target : Prends plusieurs valeurs : "_blank" pour ouvrir le lien dans une nouvelle fenêtre par exemple...

  • Lattributtitle : Permet de donner un titre au lien. Ce titre sera affiché dans une petite bulle au survol du line par la souris.
